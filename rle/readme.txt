в cmd запуск такой: node rle.js [code или decode] [файл с тестовой строкой] [файл куда сохранить результат]
в папке лежат два файла с тестовыми строками: input1.txt и input2.txt, их можно закодировать используя code, а полученный файл с результатом можно декодировать используя decode, чтобы проверить результат.
