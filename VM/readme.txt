в cmd запуск такой: node vm.js [файл для выполнения] [переменные для ввода (если есть input в файле)]
в папке лежат два файла с тестовыми программами: fib.rdl и NOK.rdl. fib.rdl считает и выводит в командную строку число фибоначчи по его порядковому номеру, который подали при запуске через командную строку. NOK.rdl считает и выводит в командную строку наименьшее общее кратное двух чисел, которые так же подали при запуске через командную строку.
